## 🔍 发现的问题

### 1. datetime序列化问题（中优先级）

**问题描述**：
- 密码重置令牌的`expires_at`字段在JSON序列化后变成字符串
- 导致datetime比较失败

**影响范围**：
- 密码重置功能
- 1个测试失败

**复现步骤**：
1. 请求密码重置
2. 保存令牌到JSON
3. 加载令牌时expires_at是字符串
4. datetime.now() < expires_at比较失败

**解决方案**：
```python
# 在service.py的reset_password方法中
from datetime import datetime

reset_token = PasswordResetToken(**token_data)
# 转换字符串为datetime
if isinstance(reset_token.expires_at, str):
    reset_token.expires_at = datetime.fromisoformat(reset_token.expires_at)
```

**修复难度**：低（5分钟）
